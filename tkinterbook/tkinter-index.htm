An Introduction to Tkinter

Note: This is the Tkinter introduction, last updated in 2005.
It was written for the outdated Python 2, and differs in a few ways from tkinter for Python 3. Some modules were renamed.

Modules renamed for Python 3:
Tkinter → tkinter
tkMessageBox → tkinter.messagebox
tkColorChooser → tkinter.colorchooser
tkFileDialog → tkinter.filedialog
tkSimpleDialog → tkinter.simpledialog
tkFont → tkinter.font
